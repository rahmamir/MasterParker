Account

Sign Up

  • The user must first click on Sign Up to add an account. The app will ask for user’s name, email, password, vehicle’s contact number and vehicle’s car plate number. Upon entering the data, the user will have the option to add credit card in which they must enter the expiry date, card number, cvv number and card name. Once the had finished creating the account, they will sign in to add new parking. The app will also provide options if the user forgets the password. Upon sign in If the data, such as, email and password matches with the email and password for sign up then the user account creation is successful. The app will navigate the user from home page which will allow the user to add parking, view the receipt and the list of their receipt for the charges made upon the monthly parking’s, provide nearby searching parking facilities, update the user’s profile by entering his/her email, the app manual for parking which provides further details and guidelines on how to use the app and the consumer support which further navigate user to an email and call dialer application

Update User Profile

  • Once the user has sign up, the app can then allow user to update their profile as name, password, contact number and car plate number. The email will not be modifying as the app will ask for that to update as followed. Once the user information has been update, the app will update the payment information if the user has added the credit card.
